claude-bar

Claude Code usage stats. Right in your menu bar.

Download for macOS

v2.0.5 · macOS 13+ · signed & notarized · ~1 MB

5h session % → projected [time to reset]

How it works

Reads your Claude session cookies straight from Chrome (decrypted via Keychain), hits the same usage endpoint claude.ai uses, caches locally. Zero login UI, zero Python, zero scripts.

Requirements

  • macOS 13 or later
  • Google Chrome with claude.ai signed in
  • Approve the Chrome Safe Storage Keychain prompt on first launch

Source & logs

Open source on GitHub. Per-fetch diagnostics land in ~/Library/Logs/claude-bar.log when something looks off.
